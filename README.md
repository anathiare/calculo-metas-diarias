# Cálculo de Metas Diárias

Olá! Bem-vinda ao **Cálculo de Metas Diárias** — uma ferramenta simples para você planejar quanto ainda precisa ganhar neste mês e descobrir **quanto precisa fazer por dia** para bater sua meta.

---

## O que este projeto faz?

Em poucas perguntas, o sistema ajuda você a:

1. **Definir sua meta** de ganho do mês
2. **Informar quantos dias** pretende trabalhar (e quantas folgas isso gera)
3. **Contar o que já fez** — dias trabalhados e lucro já obtido
4. **Mostrar o caminho** — quanto falta ganhar e o valor diário necessário

Tudo isso considerando o calendário do mês atual, suas folgas e se você está dentro ou fora do plano.

---

## Como usar

### Online (recomendado)

Acesse a página publicada no GitHub Pages:

**[Abrir Cálculo de Metas Diárias](https://anathiare.github.io/calculo-metas-diarias/)**

### No computador

1. Abra o arquivo `index.html` no navegador (Chrome, Edge, Firefox…)
2. Responda as perguntas uma a uma
3. Veja o resultado no final

Não precisa instalar nada — é HTML, CSS e JavaScript puro.
